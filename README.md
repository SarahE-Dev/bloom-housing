## Getting Started
1. **Clone the repository**
   ```sh
   git clone https://github.com/SarahE-Dev/bloom-housing.git
   cd bloom-housing
   ```
2. **Install dependencies** (if applicable)
   ```sh
   npm install
   ```
3. **Run the development server**
   ```sh
   npm run dev
   ```
   or for backend services:
   ```sh
   npm start
   ```
